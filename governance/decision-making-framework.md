Cyber crisis decisions are made under uncertainty and time pressure.
This framework ensures decisions are **structured, documented, and revisitable**.

## ðŸ§© Decision Model

For each major decision, document:

| Element | Description |
|-------|-------------|
| Decision | What needs to be decided |
| Context | Known facts and assumptions |
| Options | Available choices |
| Risks | Risks associated with each option |
| Recommendation | Preferred option |
| Decision Owner | Who decides |
| Timestamp | When the decision was made |
